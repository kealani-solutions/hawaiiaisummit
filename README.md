# Hawaii Island AI Summit 2026

Landing page for the Hawaii Island AI Summit, a hands-on AI workshop event for Hawaii's tech and business community.

**Date:** March 12, 2026
**Location:** Arc of Hilo, Hawaii

## About

The Hawaii Island AI Summit brings together leaders and builders for hands-on AI workshops, inspiring conversations, and meaningful connections.

## Development

This is a static HTML site. To run locally, simply open `index.html` in a browser or use any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## Files

- `index.html` - Main landing page
- `summit.ics` - Calendar invite file
- `images/` - Logo and image assets
